# 🚴 KEG Challenge 360 Dashboard

A mobile-first leaderboard and progress dashboard for the **KEG Challenge 360 - June Edition**.

This project tracks rider performance, weekly gains, progress toward the **360 Club**, and the race to become the **Monsoon Raja**.

---

## 🌐 Live Dashboard

**Dashboard URL**

https://sv62.github.io/keg360/

---

## 🏆 Challenge Objectives

### 🎯 360 Club

Complete **360 km** during the challenge period.

Riders who cross this milestone earn a place in the **360 Club**.

### 👑 Monsoon Raja

Complete **1000 km** during the challenge period.

The rider who reaches the target first earns the title of **Monsoon Raja**.

---

## 📊 Metrics Tracked

| Metric                   | Description                                    |
| ------------------------ | ---------------------------------------------- |
| Last Week                | Distance accumulated before the current week   |
| This Week                | Distance completed during the current week     |
| Challenge Distance       | Total challenge distance completed             |
| Distance to 360          | Remaining distance to reach the 360 Club       |
| Distance to Monsoon Raja | Remaining distance to reach the 1000 km target |

## 🔄 Updating the Dashboard

1. Update the challenge spreadsheet.
2. Generate the latest dashboard data.
3. Replace `index.html`.
4. Commit changes to the repository.
5. GitHub Pages automatically publishes the update.
